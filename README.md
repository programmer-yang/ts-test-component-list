## 这是一个基于 typescript 编写的 React 组件

这是一个简单的 List 组件

### step 1

`npm install or yarn`

### step 2

`npm build`

### step 3

`npm publish`

### Props:

| 属性 | 说明     | 类型  | 默认值 |
| ---- | -------- | ----- | ------ |
| data | 信息数据 | array | []     |

### Data:

| 属性  | 说明     | 类型   | 默认值 |
| ----- | -------- | ------ | ------ |
| key   | 唯一标识 | string | -      |
| value | 值       | string | -      |
